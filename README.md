# UNBEATABLE Landing

Static landing page with intro video -> fade -> lore sections.

## Run locally

Open `index.html` directly, or run any local static server.

## Required assets

Put files in `assets/` with exact names:

- `intro.mp4` (opening video)
- `hero_logo_1x1.png` (main logo)

Optional future assets can be added and wired in:

- `bg_graphite_texture.png`
- `shadow_double_silhouette.png`
- `beatable_panel.png`
- `unbeatable_panel.png`
- `final_banner_16x9.png`
